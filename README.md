Ez a repozitórium egy egyetemi gyakorlat SAS feladatait és megoldását tartalmazza.

Tökéletes demonstráció egy minta adatállomány generálására, majd az azon végzett statisztikai-elemzési műveletek végrehajtására.

Az adatgenerálás kódját és a gyakorló feladatokat [Csicsman József](https://math.bme.hu/~csicsman/index.html) készítette.

A feladatok megoldásához szükséges összes lekérdezést és logikát (`megoldas_lekerdezesek.sas`) önállóan fejlesztettem.
